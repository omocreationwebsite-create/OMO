Cloudinary Configuration

Configure your Cloudinary account for storing kiosk attendance photos.

Found in your Cloudinary Dashboard. This is the unique identifier for your account.

Create an unsigned upload preset in Cloudinary Settings → Upload. This allows the kiosk to upload without API keys.

How to Set Up Cloudinary

  • Go to cloudinary.com and create a free account
  • Copy your Cloud Name from the dashboard
  • Go to Settings → Upload → Upload presets → Add upload preset
  • Set Signing Mode to Unsigned
  • Set Folder to kiosk_attendance
  • Copy the preset name and paste above

Attendance Time Settings

Configure check-in and check-out times. Staff checking in late or checking out early will receive penalties.

:
AM

:
PM

Penalty Settings

Points deducted for attendance violations.

Security Settings

PIN required to exit kiosk mode and return to admin dashboard.

Kiosk Rules for Staff

  • Check-in before check-in time = ON TIME (+0 points)
  • Check-in after check-in time = LATE (-5 points)
  • Check-out after check-out time = ON TIME (+0 points)
  • Check-out before check-out time = EARLY DEPARTURE (-5 points)
  • No check-in for the day = ABSENT (-10 points)
